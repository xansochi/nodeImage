
# Give me a Cat  

Just a very simple NodeJS server with single controller.

The server is ready to be deployed on free Vercel hosting.



## Tech Stack

**Server:** Node, Express


## API ENDPOINTS
    1. '/'      - welcome

    2. '/image' - random image from folder "./images"
